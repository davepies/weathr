# Weather - a simple weather forecast thing ...

... using react and react-engine to make it *isomorph*.

![alt text][philo]

## Get it running:

```bash
npm i
npm start
# to watch
npm run watch
```
## Get tests running:
*Note: (uses zombiejs - needs iojs)*

```bash
# run them all
npm run test
# unit
npm run test:unit
# integration
npm run test:integration
```
